开发-系统开发

岗位类别

企业平台研发部（邓昀泽）

所属部门

P2-2 到 P3-1

职级范围

上海

工作地点

岗位职责：

1、参与企业平台部核心业务系统的技术规划和业务规划工作，深入理解业务需求，抽象系统模型，进行系统设计及开发工作。

2、与产品经理配合，参与需求定义，确定设计方案并负责最终实现。

3、指导低级别工程师工作和成长，带领团队进行快速的敏捷迭代开发，保证交付质量。


职位要求：

1、计算机相关专业本科以上学历，3年以上java开发工作经验，具备较强的开发能力和质量意识。

2、熟练掌握 Java 及面向对象设计开发，对部分 Java 技术有深入研究，研究过优秀开源软件的源码并有心得者优先。

3、了解 SOA 架构理念、实现技术；熟悉常见设计模式，熟练掌握 Spring、myBatis 等框架特性及应用经验。

4、熟练掌握 MySQL 应用开发、数据库原理和常用性能优化技术。

5、能够独当一面，独立负责模块/子系统的业务理解、系统设计、开发测试，运维调优。

6、有较强的逻辑/概率思维能力，善于分析、归纳、描述、沟通、和解决问题。

7、高度的创业心和投入度，既能搞定牛逼的技术难题，同时又热切地关注业务，用技术力贡献于业务成功。

8、有招聘（简历搜索、判重、推荐）相关经验者优先。
